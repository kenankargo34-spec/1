Automatically generates SEO-optimized articles to improve search rankings and increase visitor count.

== Description ==

SEO Content Generator, WordPress siteniz için otomatik olarak SEO optimize edilmiş makaleler oluşturan güçlü bir eklentidir.

Özellikler:
* OpenAI ve Google Gemini API desteği
* Toplu makale oluşturma
* Otomatik günlük içerik üretimi
* Anahtar kelime yönetimi ve önerici
* Haber kaynaklarından içerik üretme
* SEO meta verileri otomatik oluşturma
* FAQ şeması desteği
* Rank Math entegrasyonu

== Installation ==

1. Eklenti dosyalarını `/wp-content/plugins/seo-content-generator/` klasörüne yükleyin
2. WordPress admin panelinde 'Eklentiler' menüsünden eklentiyi aktifleştirin
3. 'SEO Content > API Ayarları' menüsünden API anahtarlarınızı girin
4. 'SEO Content > Anahtar Kelimeler' menüsünden anahtar kelimelerinizi ekleyin

== Frequently Asked Questions ==

= Hangi API sağlayıcıları destekleniyor? =

OpenAI (GPT-3.5, GPT-4) ve Google Gemini API'leri desteklenmektedir.

= Otomatik içerik üretimi nasıl çalışır? =

Eklenti günlük olarak belirlediğiniz anahtar kelimelerden birini seçerek otomatik makale oluşturur.

= Oluşturulan içerikler SEO optimize mi? =

Evet, tüm içerikler SEO başlıkları, meta açıklamaları ve yapılandırılmış verilerle optimize edilir.
